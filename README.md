# P-Type Differential Amplifier: Full-Custom Analog IC Design

## Project Description
Design and layout of a P-type differential amplifier with a P-type source follower output stage. The project follows the full-custom IC design flow, from transistor-level schematic simulation to physical layout and verification.

## Technical Workflow
* **Schematic & Simulation**: Transistor-level schematic design and simulation performed in LTSpice.
* **Physical Layout**: Full-custom physical layout implementation using Glade.
* **Verification**: Physical verification through Design Rule Check (DRC) and Layout Versus Schematic (LVS) to ensure a production-ready design.

## Tools Used
* **Simulation**: LTSpice
* **Layout**: Glade
* **Verification**: DRC/LVS engines
